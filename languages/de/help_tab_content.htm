AvatarPlus Einstellungen

In diesen Abschnitt gibt es zwei Optionenen. Um ein zusätzliches Feld im Kommentarbereich für die sozialen Netzwerke anzuzeigen, haken Sie die Auswahlbox Extra Feld an. Dadurch wird ein zusätzliches Feld im Kommentarbereich angezeigt in dem die Besucher einen Link zu ihren Profil in den sozialen Netzwerken Google+, Facebook oder Twitter einfügen können.
In einigen Fällen ist es jedoch ratsamer lediglich das Feld für die Homepage, welches ein Standardfeld bei WordPress ist, anzuzeigen, da so der Besucher nicht verwirrt wird wo er welche Angaben machen kann bzw. muss. AvatarPlus wird den Link im Feld Homepage auswerten und ggf. das Profilbild aus dem sozialen Netzwerk anzeigen.
Ist kein Link zu einem Profil angegeben, egal ob nun ein zusätzliches Feld genutzt wird oder nicht, so verwendet AvatarPlus den von WordPress über die E-Mail erzeugten Avatar.

AvatarPlus ist übrigens in der Lage Kurz-URLs aufzulösen wodurch die Möglichkeit besteht das ein Besucher auch eine URL wie z.B. http://www.example.com/+ als Kurz-URL zu einem Google+ Profil zu verwenden.

Um die Last bei der Anzeige der Kommentare möglichst gering zu halten, verwendet AvatarPlus einen einfachen Caching-Mechanismus (Cache = Zwischenspeicher). Dabei werden die URLs zu den Profilbildern gespeichert, nicht jedoch die Profilbilder selber. Obwohl dies eine sehr schlanke und effiziente Methode ist Daten zwischenzuspeichern, kann es sinnvoll sein von Zeit zu Zeit den Zwischenspeicher aufzuräumen.
So ist es z.B. nicht sinnvoll AvatarPlus-Daten von sehr alten Beiträgen die nur noch selten oder gar nicht mehr aufgerufen werden zu speichern. Diese Daten belegen unnötig Platz in der Datenbank und vergrößern somit z.B. ein Datenbankbackup.
AvatarPlus bietet deshalb die Möglichkeit zwischengespeicherte Daten nach einen gewissen Zeitraum automatisch zu löschen. Tragen Sie hierzu im Auswahlfeld einen Wert ein und wählen dann einen Zeitraum ( Tag(e), Woche(n), Monat(e) ) aus. Sollen die zwischengespeicherten Daten nicht gelöscht werden, so tragen Sie bitte als Wert eine Null ein, der Zeitraum ist dann unerheblich.
Bitte beachten Sie das in manchen Ländern aus rechtlichen Gründen die zwischengespeicherten Daten nach einen festgesetzten Zeitraum zwingend gelöscht werden müssen. Ebenso muss in manchen Ländern der Besucher darauf hingewiesen werden das zusätzliche Daten über ihn gespeichert werden. Sollte dies auf ihren Blog zutreffen, dann geben sie als Art der Daten bitte z.B. URL zum Profilbild des angegebenen sozailen Netzwerks und als Zeitraum für die Speicherung den von ihnen gewähleten Zeitraum, z.b. 3 Monate, an.

Google+ Einstellungen

Sie benötigen einen API-Schlüssel von Google falls Sie Profilbilder von Google+ verwenden möchten. Hier (Artikel in englischer Sprache) bekommen Sie nähere Informationen darüber wie Sie einen solchen API-Schlüssel erhalten.
Nachdem Sie einen API-Schlüssel erhalten haben, tragen Sie ihn bitte in das Eingabefeld für den Google+ API-Schlüssel ein.
